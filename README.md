# Sistema Web – Atlética Patológicos (Biomedicina UNIGRAN)

Frontend moderno para gestão de associados, carteirinha digital e parcerias da Atlética Patológicos – Biomedicina UNIGRAN (Dourados).

## Tecnologias

- React + Vite + TypeScript
- React Router
- Tailwind CSS

## Scripts principais

- `npm install` – instala as dependências
- `npm run dev` – executa o servidor de desenvolvimento
- `npm run build` – gera build de produção
- `npm run preview` – pré-visualiza o build

## Estrutura de pastas

- `src/pages` – telas principais (Login, Dashboard, Carteirinha, Parcerias, Admin)
- `src/layouts` – layouts reutilizáveis (principal e administrativo)
- `src/components` – componentes reutilizáveis (UI e cards)

## Logo da Atlética

Coloque o arquivo do mascote em:

- `public/imagens/logo-biomed.png`
